Pluto is the most distant planet in the solar system. This is what was written on the pages of the book earlier. On August 24, 2006, Pluto lost its status as a planet. This is because the earlier formal definition of a planet has been changed. Pluto is no longer included as a planet because it does not match the characteristics specified in the new definition. Pluto has since been called a minor planet or dwarf planet.<\/p>\n\n\n\n
